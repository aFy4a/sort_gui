# Целые числа меньше 6

Данный пакет принимает на вход массив целых чисел, выводит массив, в котором числа меньше 6.


Клонирование репозиторя;
```bash
git clone https://github.com/aFy4a/sort_gui.git
cd sort
poetry install
```

Запуск:
```bash
poetry run sort
```

Тесты:
```bash
poetry run pytest
```
